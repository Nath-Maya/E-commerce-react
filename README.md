# E-commerce Wonder Bike 💰

## 🚨 🚧 Proyecto en proceso...🚨 🚧 

Programa donde el usuario/cliente puede seleccionar elementos o articulos que desea comprar, agregandolos al carrito de la pagina. Podra visualizar el listado de los productos con su respectivo precio y el total de la compra.  

Es un e-commerce enfocado en productos para un publico objetivo, como personas que practican el deporte : Ciclismo.
